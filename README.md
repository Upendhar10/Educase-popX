## EDUCASE Frontend Dev Assignment:

### POPX 
A React-based application showcasing user registration, login authentication, and account management, with user information stored in **localStorage** as an array.

### Features

* User signup with form validation
* Login verification using saved user credentials
* Persistent data storage through localStorage
* Dynamic rendering of account information
* Responsive and clean interface built with Tailwind CSS
* Routing and navigation powered by React Router

### Usage

* Register a new account through the signup form; user details are saved in localStorage.
* Login with your registered email and password.
* Access account settings dynamically populated from stored user information.

### Technologies Used

* React + Typescript
* React Router
* Tailwind CSS
* react-toastify for toast notifications
* Browser localStorage for data persistence

### Notes

* Passwords are currently stored in localStorage. For real-world applications, use secure authentication and storage practices.
* The project is designed to demonstrate frontend functionality without backend implementation.
